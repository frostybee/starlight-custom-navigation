The tables below include some SVG paths that can be used for the navigation buttons in the custom navigation plugin.

:::tip
You can customize the appearance of the navigation buttons by changing the **`path`** and **`strokeWidth`** options in the configuration object passed to the plugin. 
:::

:::note
The navigation buttons will **automatically rotate** the SVG path 180 degrees for the "**next**" button, so you **only need** to provide the path for the "previous" button.
:::

## Configuration Example

The example below shows how to **use these paths** in your code:
```javascript title="astro.config.mjs"
export default defineConfig({
   integrations: [
     starlight({
       plugins: [
        starlightCustomNavigation({ 
            buttonIcon: {
                path: 'M4 16L12 8L20 16',
                strokeWidth: 2
            }
        })
       ]
     })
   ]
 });
```

### Customizing Stroke Width

You can adjust the thickness of the navigation arrows by modifying the `strokeWidth` property. Here are some examples:

```javascript
// Thin stroke
buttonIcon: {
    path: 'M4 16L12 8L20 16',
    strokeWidth: 1
}

// Medium stroke (default)
buttonIcon: {
    path: 'M4 16L12 8L20 16',
    strokeWidth: 2
}

// Thick stroke
buttonIcon: {
    path: 'M4 16L12 8L20 16',
    strokeWidth: 3
}
```
